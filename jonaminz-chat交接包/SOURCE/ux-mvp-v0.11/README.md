# Jonaminz Chat + Shared MVP v0.11

這是一份可直接下載、解壓縮並雙擊開啟的前端互動原型。

## 開啟方式

1. 解壓縮 ZIP。
2. 雙擊 `index.html`。
3. 建議用 Chrome 或 Edge 開啟。
4. 不需要安裝、不需要伺服器、不會連到正式資料庫。

## 可以測試的功能

- 右下角聊天泡泡
- 邊看頁面邊聊天
- Jonathan / Minz 身分切換
- 一般文字與簡單貼圖
- 傳送目前頁面
- 「截取目前畫面」卡片示意
- OneDrive / Google Drive 附件來源示意
- Shared 未一起看到
- 分享卡進入視野 1 秒後自動標為已看到
- 標回未看
- 新增網址
- 同網址去重與合併
- 表情反應
- 加入 Movie / Travel
- 淺色 / 深色 Theme
- localStorage 保存狀態
- 重設 Demo

## 重要說明

這是 UX 與架構 MVP，不是正式聊天產品。

目前沒有：

- 真實聊天同步
- Supabase
- OneDrive 登入與上傳
- Android 泡泡
- Android Share Target
- 真實截圖
- 正式 Jonaminz Core 接入

完整架構決策請看 `CHAT_SHARED_ARCHITECTURE.md`。


## v0.2 修正

- 移除內容頁上的「傳送目前頁面」按鈕；此功能只保留在聊天工具列。
- 修正 Shared 卡片被 CSS Grid 壓縮、內容遭裁切的問題。
- 修正所有卡片同時進入可視範圍，造成未看數量瞬間歸零的問題。
- 改善手機版 Shared 卡片高度、文字與操作按鈕排版。
- 自動「已看到」只在 Shared 分頁真正開啟且卡片可見時執行。


## v0.3 改善

- 聊天視窗開啟後隱藏右下角泡泡，新增最小化。
- 聊天視窗提供小／中／大三段尺寸。
- 聊天首頁新增 Shared 未一起看到摘要。
- Shared 顯示「你尚未看到／對方尚未看到／尚未一起看到」三組數量。
- Shared 卡片改成緊湊橫式，一個畫面可看到更多內容。
- 卡片表面只保留愛心、笑臉、討論與更多；Movie、Travel、未看等收入 `⋯`。
- Composer 改為 `＋` 選單，並把「分享目前內容」列為主要動作。
- 點「討論」後建立綁定狀態，後續訊息會顯示回覆哪一筆分享。
- 自動「已看到」加入焦點、滑動速度、72% 可見與 1.2 秒停留條件。
- 手動標回未看後，8 秒內不會立刻被自動標回已看到。


## v0.4 改善

- 右下角聊天泡泡永遠保留；同一顆泡泡負責展開與縮小。
- Shared 統計卡改成可按篩選：
  - 你尚未看到
  - 對方尚未看到
  - 尚未一起看到
  - 全部分享
- 尚未看到的卡片顯示紅點；已看到後只移除紅點，不降低整張卡片透明度。
- 聊天標題列新增電話按鈕。
- 電話功能在 MVP 中模擬直接撥號與通話狀態；正式 Android App 將透過 Capacitor 原生 plugin 接 Android `ACTION_CALL`。


## v0.5 修正

- 移除卡片進入視野後自動標記「已看到」的邏輯。
- 紅點會一直保留，直到使用者真正點進該分享卡片。
- 點卡片會開啟詳細預覽，並在此時標記為「已看到」。
- 點「討論」或「開啟原始內容」也視為真正進入內容，因此會標記已看到。
- 詳細預覽中仍可「標回未看」。


## v0.6 修正

- 右下角聊天入口不再顯示聊天符號或縮小符號。
- 入口永遠代表對方：
  - Jonathan 身分顯示 `M`
  - Minz 身分顯示 `J`
- 展開與縮小聊天時，對方頭像不改變。
- 未讀數直接掛在對方頭像右上角。
- 正式版可直接將字母 avatar 替換成兩人的真實照片。


## v0.7 改善

- 移除無實際用途的小型聊天面板。
- 移除標題列的 `—` 與 `×`，避免和右下角對方頭像功能重複。
- 狀態只剩三種：
  1. 對方頭像（收合）
  2. 半版聊天室（日常使用）
  3. 全版聊天室（搜尋、整理分享、檔案）

- 加入 Chat Library instance 概念，為未來 SKHPSv2 獨立聊天室預留正確架構。
- 新增 `TEST_CHECKLIST.md`。


## v0.8 改善

- 手機全版聊天室改為 Messenger 式沉浸介面。
- 頂部固定對方大頭貼、姓名、在線狀態示意。
- 新增電話、視訊、聊天資訊與半版／全版切換。

- 分享內容卡片改成更接近聊天訊息中的媒體卡。
- 新增聊天資訊頁，可從同一處進入電話、視訊與檔案。
- 手機全版使用獨立深色 Chat Theme tokens，不改動原頁面 Theme。


## v0.9 改善

- 聊天右上角只保留電話。
- 視訊與資訊按鈕先移除；未來只有在真正啟用 provider 時才顯示。
- 半版／全版切換移到頂部拖曳把手，避免右上角堆滿工具。
- Shared 卡片重新設計：來源、網域、分享者、標題、附言、未看與目的地資訊分層顯示。
- Shared 卡片不再固定放愛心、笑臉、Movie、Travel 等一排按鈕。
- `送往…` 改為 registry-driven 目的地選擇器；未來增加專案不會讓卡片越來越長。
- 輸入框空白時，右側按鈕為可自訂快速心情；有文字時自動變為送出。
- 長按快速心情按鈕可改成其他符號。
- 貼圖移到輸入框旁的獨立按鈕。
- 長按聊天文字或分享卡片可加入／更換心情反應。


## v0.10 改善

- 修正心情列位置：長按哪一則訊息，就貼著該訊息出現，不再跑到整個頁面底部。
- 桌機 hover 訊息時顯示三顆按鈕：回應、回覆、更多。
- 新增訊息回覆綁定與更多選單。
- 表情符號按鈕移入文字輸入框，可直接插入游標位置。
- 貼圖改放進 `＋` 選單，仍以大型貼圖訊息送出。
- Shared「全部分享」依 `lastViewedAt` 排序，剛看過的內容回到最上面。
- Shared 記錄 `sharedAt` 與 `lastViewedAt`。
- 「送往…」新增搜尋，可支援未來更多註冊目的地。
- Launcher badge 改為真正的未讀訊息數，不再拿 Shared 未看數代替。
- 新增輸入中、未讀分隔線、最後訊息時間與已讀頭像的 MVP 狀態模型。
